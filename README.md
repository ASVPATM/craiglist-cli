**Simple Craigslist CLI for ease of use**

**List of World Cities came from https://github.com/datasets/world-cities.git**

# Setup
Clone The Directory

**Python3**
```
cd craiglist-cli; cd craig
```
**Install Dependencies**
```
pip3 install -r requirements.txt
```
**Run Main Script**
```
python3 script.py
```
